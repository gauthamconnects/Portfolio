Setting up the site

November 16, 2023 2-minute read
Gauthamkrishna N

As I didn’t have any prior knowledge about hugo my first step was to get to know about hugo and how it works before using it to build a website as the task asks me to do.So I googled about hugo,visited the official site and saw a few Youtube videos of using hugo themes to build a portfolio site.

So now I had to install hugo into my system and used “sudo apt-get install git” to install hugo using the terminal.Then I went through some of the themes in hugo’s themes library and downloaded the “hugo-coder” theme to base my portfolio site on.I downloaded the zip file from the git repository and extracted the zip file in the system.Then I made a copy of the exampleSite folder in the extracted file and moved the hugo-coder file into the themes folder of the exampleSite folder.I did renamed the extracted folder to coder and then edited the “config.toml” file in the root directory of exampleSite and gave themes as coder as the folder inside themes was named.

Then I used “hugo server” command to preview the site locally.After using that command the site was locally set up to be previewed at “http://localhost:1313/” and now I could see the example site and I was able to make changes on it and make it the way I wanted my site to look like.

So now that’s almost it for setting up the site and I moved on to design the site as per my need.
