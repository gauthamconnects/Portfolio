Designing The Site

November 16, 2023 3-minute read
Gauthamkrishna N

So now that the site is set up locally to be designed as per my wish I did start making changes and the main page of the site looked like this at first.

Main Page Link to heading

First of all I decided to make changes in the main page.To do that I opened “config.toml” in the root directory then changed the title,author,description,keywords,info etc.. and I also replaced the avatar image with a picture of myself after adding my picture to the images folder in the static directory and mentioning the path to the image in “config.toml”.Then I removed the alternate language given in the site as I only wanted it to be in English.

Now I had to add Icons that I want in the main page that redirects to my accounts onclick.Since I wasn’t a user of some of the platforms that were already there in the page I had to delete some of them like gitlab,medium etc and added instagram instead.I had to add the awesome font icon of instagram aswell since it wassn’t already there.Now that the icons are set up I had to link it to my accounts so I changed the URLs to the respective accounts I have in each platform and that was probably it for the main page.

Contact Page Link to heading

Now I had to change the other pages of the site and I just changed Contact me header to Contact with I thought was better.There wasn’t much content there in the exampleSite so I just added the platforms and the respective URLs to the platforms.I did align the data into a table as well to look more organised and that was it for the Contact Page.

Blog Page Link to heading

So there were several posts in this page but I only needed 3 to write this blog as I divided the entire process of doing this task into 3.First one was setting up the site,second one was desinging it and the third one was hosting it.

So I removed the rest of the posts and renamed the the posts to Setting up the site,Designing the site and Hosting the site.I also changed the date to the date when I worked on this blog.Then I went on to edit the pages for the respective posts.I kept the title but changed it according to the post’s content,kept the date and changed the author’s name to my name itself.Then I just wrote in detail about all of the work I have done to complete this task.

About Page Link to heading

Here I hadn’t changed much becuase there was hardly any content in there so I just wrote about myself and my aspirations in that page.

Projects Page Link to heading
