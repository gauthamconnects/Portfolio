Hosting the site

November 17, 2023 One-minute read
Gauthamkrishna N

So the site is designed as I wanted and now its time to host it on github pages and make it public so that people can access it.

I created a github repository named Portfolio so that the necessary files can be uploaded there to host the site.Before that I have to convert the markdown(.md) files into HTML,JavaScript,CSS etc.. to do that I gave “canonifyURLs=true” in the “config.toml” file of the root directory.Then I changed the baseURL to “https://gauthamconnects.github.io/Portfolio/".

Then I used “hugo -D” command to build the site.After that there was a “public” directory in the root directory and I went into that directory to upload the files in that directory to the github repository.For that I initialised using “git init” then used “git remote add origin https://github.com/gauthamconnects/Portfolio.git" to set up the repository.Then used “git add .” to stage all the files,“git commit -m"first commit” to commit the changes and pushed the files using “git push”.

Now that the required files are in the repository,from the settings tab of the repository I went to the Pages option and selected the “main” branch as the source.Then gave the domain name and the site was set live.
